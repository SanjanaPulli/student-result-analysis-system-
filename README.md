# Student Result Analysis System

A comprehensive web-based result management system that automates student performance analysis and grading processes.

## Features
- Automated CSV upload for bulk student data
- Interactive analytics dashboard with Chart.js
- Grade distribution visualization
- Role-based access control

## Tech Stack
- **Backend**: Flask, Python
- **Database**: MySQL
- **Frontend**: HTML, CSS, Bootstrap, Chart.js
- **Data Processing**: Pandas
